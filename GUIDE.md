## 📋 О приложении

**Bestlight Monitor** — это веб-приложение для мониторинга данных из Google Sheets в реальном времени.

### Основные возможности:

- ✅ Авторизация с безопасным хранением сессии
- ✅ Просмотр данных из Google Sheets
- ✅ Поиск по всем полям таблицы
- ✅ Сортировка по любой колонке
- ✅ Автоматическая пагинация (при >200 записей)
- ✅ Кэширование данных (5 минут)
- ✅ Адаптивный дизайн для мобильных устройств

---

## 🚀 Начало работы

### Системные требования

- Любой современный браузер (Chrome, Firefox, Safari, Edge)
- Доступ к интернету
- Действующие учетные данные для входа

### Первый вход

1. Откройте приложение по адресу: **https://bestlight-monitor.vercel.app**
2. Введите логин и пароль, предоставленные администратором
3. Нажмите кнопку **«Войти»**

## 🔐 Вход в систему

### Форма входа

![Форма входа](./docs/login-form.png)

**Поля формы:**

| Поле | Описание |
|------|----------|
| **Логин** | Ваше имя пользователя (например, `admin`) |
| **Пароль** | Ваш секретный пароль |

### Ошибки при входе

| Сообщение | Причина | Решение |
|-----------|---------|---------|
| «Неверный логин или пароль» | Неверные учетные данные | Проверьте правильность ввода |
| «Ошибка при входе» | Техническая ошибка | Попробуйте позже или обратитесь в поддержку |

### Безопасность

- Пароль хешируется алгоритмом **SHA-256** перед отправкой
- Сессия хранится в **sessionStorage** браузера
- При закрытии вкладки требуется повторный вход

## 🔍 Поиск и фильтрация

### Поиск по данным

1. Введите поисковый запрос в поле **«Поиск по всем полям»**
2. Результаты фильтруются автоматически
3. Для сброса нажмите **✕** в поле поиска

![Поиск](./docs/search.png)

### Особенности поиска

- Поиск **нечувствителен к регистру** (Admin = admin)
- Ищет совпадения **во всех полях** одновременно
- Поддерживает **частичные совпадения** (BMW найдет BMW X5)

## ↕️ Сортировка данных

### Сортировка по колонкам

1. Кликните на **заголовок колонки** для сортировки
2. Первый клик — **по возрастанию** (↑)
3. Второй клик — **по убыванию** (↓)

![Сортировка](./docs/sort.png)

### Индикаторы сортировки

| Индикатор | Значение |
|-----------|----------|
| **↑** | Сортировка по возрастанию |
| **↓** | Сортировка по убыванию |
| **Нет индикатора** | Сортировка не применена |

### Умная сортировка

- **Числовые данные** сортируются как числа (10 > 2)
- **Текстовые данные** сортируются по алфавиту
- **Пустые значения** отображаются в конце

## 📄 Пагинация

### Автоматическая пагинация

Пагинация включается автоматически при **>200 записей** в таблице.

![Пагинация](./docs/pagination.png)

### Элементы управления

| Элемент | Описание |
|---------|----------|
| **← Назад** | Переход на предыдущую страницу |
| **1 2 3 ...** | Прямой переход на страницу |
| **Вперёд →** | Переход на следующую страницу |
| **Страница X из Y** | Текущая позиция |

### Навигация

- Кликните на **номер страницы** для прямого перехода
- Используйте кнопки **«Назад»/«Вперёд»** для последовательной навигации
- При поиске пагинация обновляется автоматически

## 🔄 Обновление данных

### Ручное обновление

1. Нажмите кнопку **«🔄 Обновить»** в правом верхнем углу
2. Дождитесь загрузки данных
3. Проверьте время последнего обновления

![Обновление](./docs/refresh.png)

### Автоматическое кэширование

- Данные кэшируются на **5 минут**
- При повторном открытии показываются закэшированные данные
- Кнопка «Обновить» загружает свежие данные

### Индикаторы состояния

| Индикатор | Значение |
|-----------|----------|
| **🔄 Обновить** | Данные загружены |
| **Спиннер** | Идёт загрузка |
| **«Последнее обновление: XX:XX»** | Время загрузки |

## ❓ Техническая поддержка

### Частые проблемы

#### «Данные не загружаются»

**Причины:**
- Нет доступа к интернету
- Неверный API ключ Google Sheets
- Таблица недоступна

**Решение:**
1. Проверьте подключение к интернету
2. Обновите страницу (F5)
3. Обратитесь к администратору

#### «Неверный логин или пароль»

**Причины:**
- Неправильный ввод
- Изменились учетные данные

**Решение:**
1. Проверьте раскладку клавиатуры
2. Убедитесь, что Caps Lock выключен
3. Запросите новые данные у администратора

#### «Таблица отображается некорректно»

**Причины:**
- Проблемы с браузером
- Устаревший кэш

**Решение:**
1. Очистите кэш браузера (Ctrl+Shift+Delete)
2. Попробуйте другой браузер
3. Обновите страницу
